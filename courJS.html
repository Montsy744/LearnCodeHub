Introduction au JavaScript

    JavaScript est souvent utilisé pour rendre les pages web interactives. Voici un exemple simple de code JavaScript :

                    <!DOCTYPE html>
                    <html>
                        <head>
                            <title>Exemple JavaScript</title>
                        </head>
                        <body>
                            <h1>Bonjour, Monde !</h1>
                            <button> onclick="saluer()">Cliquez-moi</button>

                            <script>>
                                function saluer() {
                                    alert('Bonjour à tous!');
                                }
                            </script>
                    </body>
                    </html>
