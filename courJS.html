Introduction au JavaScript

    JavaScript est souvent utilisé pour rendre les pages web interactives. Voici un exemple simple de code JavaScript :

                    <!DOCTYPE html>
                    <html>
                        <head>
                            <title>Exemple JavaScript</title>
                        </head>
                        <body>
                            <h1>Bonjour, Monde !</h1>
                            <button> onclick="saluer()">Cliquez-moi</button>

                            <script>>
                                function saluer() {
                                    alert('Bonjour à tous!');
                                }
                            </script>
                    </body>
                    </html>

Dans cet exemple, lorsque vous cliquez sur le bouton, une alerte s'affiche avec le message "Bonjour à tous!".

Les bases du JavaScript

Voici quelques concepts de base en JavaScript :

Variables : Elles permettent de stocker des données.

                        var nom = "Alice";
                        let age = 25;
                        const ville = "Paris";

                        let nombre = 42; // Nombre
                        let texte = "Salut!"; // Chaîne de caractères
                        let vraiFaux = true; // Booléen
                        let tableau = [1, 2, 3]; // Tableau
                        let objet = { nom: "Jean", age: 30 }; // Objet

Fonctions : Elles sont des blocs de code réutilisables.

                        // Déclaration de la fonction
                        function saluer(nom) {
                        return `Bonjour, ${nom}!`;
                        }

                        // Utilisation de la fonction
                        const message = saluer("Alex");
                        console.log(message); // Affiche "Bonjour, Alex!"

Conditions : Elles permettent d'exécuter du code sous certaines conditions.

                        if (age > 18) {
                            console.log("Adulte");
                        } else {
                            console.log("Mineur");
                        }

Boucles : Elles permettent de répéter des actions.

                        for (let i = 0; i < 5; i++) {
                            console.log(i);
                        }

Le DOM

    La manipulation du DOM (Document Object Model) est une partie essentielle de JavaScript, surtout pour rendre les pages web interactives. Voici un développement plus détaillé sur ce sujet.

Qu'est-ce que le DOM ?

Le DOM est une représentation en arbre de la structure d'un document HTML ou XML. Chaque élément du document devient un "nœud" dans cet arbre, ce qui permet à JavaScript de naviguer et de manipuler ces nœuds.

Naviguer dans le DOM

Il est possible de naviguer dans l'arborescence du DOM pour atteindre différents éléments.

Accéder aux parents, enfants et frères/sœurs

                        let parent = element.parentNode;  // Accède au parent de l'élément
                        let children = element.childNodes;  // Accède à tous les enfants de l'élément (nœuds texte inclus)
                        let firstChild = element.firstChild;  // Accède au premier enfant
                        let lastChild = element.lastChild;  // Accède au dernier enfant
                        let nextSibling = element.nextSibling;  // Accède au prochain frère/sœur
                        let previousSibling = element.previousSibling;  // Accède au frère/sœur précédent

Sélection des éléments du DOM

La première étape pour manipuler le DOM est de sélectionner les éléments que tu veux modifier. Voici quelques méthodes pour le faire :

                        // Sélection par ID
                        let element = document.getElementById('monId');

                        // Sélection par classe
                        let elements = document.getElementsByClassName('maClasse');

                        // Sélection par balise
                        let tags = document.getElementsByTagName('p');

                        // Sélection avec sélecteurs CSS
                        let query = document.querySelector('.maClasse');
                        let allQueries = document.querySelectorAll('.maClasse');

Manipulation des éléments du DOM

    Une fois que tu as sélectionné un élément, tu peux le manipuler de différentes manières :

Changer le contenu

                        element.innerHTML = 'Nouveau contenu';  // Permet d'inclure du HTML
                        element.textContent = 'Nouveau texte';  // Juste du texte

Changer les styles

                        element.style.color = 'blue';  // Change la couleur du texte
                        element.style.backgroundColor = 'yellow';  // Change la couleur de fond
                        element.style.fontSize = '20px';  // Change la taille de la police

Ajouter/Supprimer des classes

                        element.classList.add('nouvelleClasse');  // Ajoute une classe
                        element.classList.remove('ancienneClasse');  // Supprime une classe
                        element.classList.toggle('classe');  // Ajoute ou supprime une classe selon son état

Créer et ajouter des éléments

                        let newElement = document.createElement('div');  // Crée un nouvel élément <div>
                        newElement.textContent = 'Contenu du nouvel élément';  // Ajoute du contenu
                        document.body.appendChild(newElement);  // Ajoute le nouvel élément à la fin du <body>

Supprimer des éléments

                        element.remove();  // Supprime l'élément sélectionné

Les attributs HTML

    Grace au JavaScript il est possible de modifier, récupérer ou suprimer la valeurs d'un attributs HTML

Travailler avec les Attributs

Les attributs des éléments HTML peuvent être manipulés avec JavaScript.

Obtenir et définir des attributs

                        let attribut = element.getAttribute('src');  // Obtenir la valeur d'un attribut
                        element.setAttribute('src', 'nouvelleValeur');  // Définir une nouvelle valeur pour un attribut
                        element.removeAttribute('src');  // Supprimer un attribut

Exemples Pratiques

Voici quelques exemples pratiques pour illustrer comment manipuler le DOM :

Exemple 1 : Ajouter un nouvel élément avec du texte

                        let newParagraph = document.createElement('p');
                        newParagraph.textContent = 'Ceci est un nouveau paragraphe.';
                        document.body.appendChild(newParagraph);

Exemple 2 : Modifier le style d'un élément quand il est cliqué

                        let button = document.getElementById('monBouton');
                        button.addEventListener('click', function() {
                            button.style.backgroundColor = 'green';
                        });

Les événements en JavaScript

    En JavaScript, un événement est une action ou une occurrence détectée par le navigateur, à laquelle on peut attacher une réponse. Les événements peuvent être déclenchés par diverses actions de l'utilisateur, telles que :

  • Cliquez sur un élément :par exemple, un bouton ou un lien.

  • Mouvement de la souris :par exemple, lorsqu'un utilisateur déplace la souris sur un élément.

  • Pression d'une touche :par exemple, lorsqu'un utilisateur appuie sur une touche du clavier.

  • Modification d'un champ de formulaire :par exemple, lorsqu'un utilisateur entre du texte dans un champ de saisie.

Écouteurs d'Événements

Les événements permettent de rendre une page interactive en réagissant aux actions de l'utilisateur (clics, survols, etc.).

Comment gérer les événements ?

JavaScript offre plusieurs façons de gérer les événements :

  • Directement dans le HTML 
                                    <button onclick="alert('Bouton cliqué !')">Cliquez-moi</button>
  • Avec la méthode addEventListener() 
                                    document.getElementById("monBouton").addEventListener("click", function() {
                                    alert('Bouton cliqué !');
                                  });
  • Utilisation de fonctions anonymes ou nommées 
                                    function afficherMessage() {
                                    alert('Bouton cliqué !');
                                }
                                  
                                document.getElementById("monBouton").addEventListener("click", afficherMessage);

Événements communs :

  1. click : Quand un élément est cliqué.

  2. mouseover : Quand la souris passe sur un élément.

  3. mouseout : Quand la souris quitte un élément.

  4. keydown : Quand une touche du clavier est enfoncée.

Manipulation des Evénements Avancée

Vous pouvez ajouter des gestionnaires d'événements pour des interactions plus complexes :

                        document.querySelector('button').addEventListener('mouseover', function() {
                            console.log('Le bouton est survolé.');
                        });
                        
                        document.querySelector('button').addEventListener('click', function() {
                            console.log('Le bouton est cliqué.');
                        });

L'API Fetch

Qu'est-ce que l'API Fetch?

L'API Fetch est une interface moderne et flexible pour effectuer des requêtes HTTP en JavaScript. Elle est utilisée pour interagir avec des serveurs afin de récupérer ou envoyer des données. Contrairement à l'ancienne méthode XMLHttpRequest, l'API Fetch est plus simple à utiliser et offre des fonctionnalités avancées comme les Promises, facilitant la gestion des requêtes asynchrones.

Principaux avantages de l'API Fetch :

Simplicité :Syntaxe claire et concise.

Promesses :Utilise des Promises pour gérer les opérations asynchrones, rendant le code plus lisible.

Supporte CORS : Gère les requêtes entre différentes origines de manière sécurisée.

Utilisation de l'API Fetch

L'API Fetch permet de récupérer des ressources à partir d'un serveur. Voici un exemple de requête GET :

                        fetch('https://api.example.com/donnees')
                            .then(response => response.json())
                            .then(data => {
                                console.log(data);
                            })
                            .catch(error => {
                                console.error('Erreur:', error);
                            });
