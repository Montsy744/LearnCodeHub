Introduction au CSS

CSS est une technologie utilisée pour ajouter du style aux pages web. Il permet de contrôler la mise en page, les couleurs, les polices, les espaces, et bien plus encore. CSS est séparé du HTML, ce qui permet de maintenir un code HTML propre et de styler plusieurs pages avec un seul fichier CSS.

Syntaxe de base

Un fichier CSS est composé de règles. Chaque règle a deux parties principales : un sélecteur et une déclaration. Une déclaration comprend une propriété et une valeur.

                        sélecteur {
                            propriété: valeur;
                        }

Exemple :

                        h1 {
                            color: blue;
                            font-size: 20px;
                        }

Inclusion du CSS dans un Document HTML

Feuille de Style Interne :

                        <style>
                            p {
                              color: blue;
                            }
                        </style>

Explication : Cet exemple montre comment ajouter du CSS directement dans un document HTML à l'aide de la balise <style>. Ici, tous les paragraphes auront leur texte en bleu.

Feuille de Style Externe :

                        <link rel="stylesheet" href="styles.css">

Explication : Vous pouvez également inclure une feuille de style externe en utilisant la balise <link>. Cela rend votre code plus propre et permet de réutiliser les styles dans plusieurs documents.

Attribut de Style En-Ligne :

                        <p style="color: green;">Ce texte est vert.</p>

Explication : L'attribut style permet d'ajouter du CSS directement dans une balise HTML. Dans cet exemple, le texte du paragraphe sera vert.

Sélecteurs

Les sélecteurs permettent de cibler des éléments HTML spécifiques à styler.

Sélecteurs de base

Sélecteur de type : cible tous les éléments du même type.

                        p {
                            color: green;
                        }

Sélecteur de classe : cible tous les éléments avec une certaine classe. Utilisez un point (.) avant le nom de la classe.

                        .maClasse {
                            color: red;
                        }

Sélecteur d'identifiant : cible un élément avec un certain identifiant. Utilisez le signe dièse (#) avant l'identifiant.

                        #monID {
                            color: purple;
                        }

Propriétés CSS communes

Couleurs et arrière-plans

color: Définit la couleur du texte.

                        p {
                            color: blue;
                        }

background-color: Définit la couleur de l'arrière-plan.

                        div {
                            background-color: yellow;
                        }

Polices et texte

font-family: Définit la police du texte.

                        h1 {
                            font-family: Arial, sans-serif;
                        }

font-size: Définit la taille du texte.

                        p {
                            font-size: 16px;
                        }

font-weight: Définit la graisse de la police (normal, bold, etc.).

                        h2 {
                            font-weight: bold;
                        }

Boîte de modèle

width et height: Définit la largeur et la hauteur d'un élément.

                        div {
                            width: 100px;
                            height: 50px;
                        }

margin: Définit la marge extérieure d'un élément.

                        p {
                            margin: 20px;
                        }

padding: Définit la marge intérieure d'un élément.

                        div {
                            padding: 10px;
                        }

border : Définit la bordure d'un élément.

                        img {
                            border: 1px solid
                        }

Sélecteurs de Groupe et Combinés

Sélecteurs de Groupe:

                        h1, h2, h3 {
                            color: darkred;
                        }

Explication : Ce sélecteur de groupe cible toutes les balises <h1>, <h2>, et <h3> et change leur couleur de texte en rouge foncé.

Sélecteurs Descendants:

                        div p {
                            color: green;
                        }

Explication : Ce sélecteur descendant cible tous les paragraphes

qui sont des enfants directs ou indirects des éléments <div> et change leur couleur de texte en vert.

Concepts Avancés du CSS

Concepts Avancés de Sélecteurs CSS

Sélecteurs d'Attributs

[attr=value]: Sélectionne les éléments dont l'attribut attr a exactement la valeur value.

                        input[type="text"] {
                            border: 1px solid #000;
                        }

[attr~=value]: Sélectionne les éléments dont l'attribut attr contient la valeur value en tant que mot unique.

                        [class~="primary"] {
                            color: blue;
                        }

[attr|=value]: Sélectionne les éléments dont l'attribut attr a exactement la valeur value ou commence par value suivi d'un tiret (-).

                        a[lang|="en"] {
                            font-style: italic;
                        }

[attr^=value]: Sélectionne les éléments dont l'attribut attr commence par la valeur value.

                        a[href^="https"] {
                            color: green;
                        }

[attr$=value]: Sélectionne les éléments dont l'attribut attr se termine par la valeur value.

                        img[src$=".jpg"] {
                            border: 1px solid #ccc;
                        }

[attr=value]:* Sélectionne les éléments dont l'attribut attr contient la valeur value.

                        a[href*="example"] {
                            background-color: yellow;
                        }

Pseudo-Classes et Pseudo-Éléments Avancés

:nth-child(n): Sélectionne le nième enfant de son parent.

                        li:nth-child(2) {
                            color: red;
                        }

:nth-of-type(n): Sélectionne le nième enfant de son parent, parmi les éléments du même type.

                        p:nth-of-type(2) {
                            font-weight: bold;
                        }

:not(selector): Sélectionne les éléments qui ne correspondent pas au sélecteur.

                        p:not(.highlight) {
                            color: gray;
                        }

::before et ::after: Ajoute du contenu avant ou après le contenu d'un élément.

                        p::before {
                            content: "Note: ";
                            font-weight: bold;
                        }
                        p::after {
                            content: " [fin]";
                        }

Mise en Page avec Flexbox

Bases de Flexbox

Conteneur flex: Crée un conteneur flexible (display: flex).

                        .container {
                            display: flex;
                        }

Axe principal et axe transversal: Concepts de direction dans un conteneur flex.

                        .container {
                            flex-direction: row;     /* axe principal horizontal */
                            flex-direction: column;  /* axe principal vertical */
                        }

Alignement et Justification

Alignement des éléments sur l'axe transversal:

                        .container {
                            align-items: center;
                            align-items: flex-start;
                            align-items: flex-end;
                        }

Justification des éléments sur l'axe principal:

                        .container {
                            justify-content: center;
                            justify-content: space-between;
                            justify-content: space-around;
                        }

Ordre et Orientation

Ordre des éléments:

                        .item {
                            order: 2;
                        }

Orientation du conteneur:

                        .container {
                            flex-wrap: wrap;
                            flex-direction: row-reverse;
                        }

Mise en Page avec Grid Layout

Concepts de Base de Grid Layout

Définir une grille:

                        .grid-container {
                            display: grid;
                            grid-template-columns: repeat(3, 1fr);
                            grid-template-rows: auto;
                        }

Positionnement des Éléments

Définir les colonnes et les lignes des éléments:

                        .item {
                            grid-column: 1 / 3;
                            grid-row: 2 / 4;
                        }

Alignement et Espacement

Aligner les éléments dans la grille:

                        .grid-container {
                            justify-items: center;
                            align-items: stretch;
                        }

Variables CSS (Custom Properties)

Déclaration et Utilisation des Variables

Déclarer une variable:

                        :root {
                            --main-color: #3498db;
                        }

Utiliser une variable:

                        .element {
                            color: var(--main-color);
                          }

Scope et Héritage

Utiliser des variables dans différents contextes:

                        .container {
                            --main-color: #e74c3c;
                        }
                          .item {
                            color: var(--main-color);
                        }
