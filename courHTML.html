Introduction a l'HTML

    HTML (HyperText Markup Language) est le langage standard utilisé pour créer des pages web. Il est utilisé pour structurer le contenu sur le web et il est composé d'éléments qui définissent différentes parties d'une page.

1. Structure de base d'une page HTML

Chaque document HTML commence par une déclaration de type :

<!DOCTYPE html>

Cette déclaration indique au navigateur que nous utilisons HTML5. Voici la structure de base d'une page HTML :


                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Ma première page HTML</title>
                    </head>
                    <body>
                        <h1>Bienvenue dans le monde du HTML!</h1>
                        <p>Ceci est un paragraphe.</p>
                    </body>
                    </html>

La partie <head>

    La balise <head> contient des méta-informations sur le document. Ces informations ne sont pas affichées directement sur la page mais sont cruciales pour les moteurs de recherche, les navigateurs et autres services web. Voici quelques éléments courants dans la balise <head> :


<title> : Le titre de la page, affiché dans l'onglet du navigateur.


                    <title>Ma première page HTML</title>

<meta> : Les balises meta fournissent des informations telles que le jeu de caractères utilisé, l'auteur de la page, ou des mots-clés pour les moteurs de recherche. 


                    <meta charset="UTF-8">
                    <meta name="description" content="Une introduction au HTML">
                    <meta name="keywords" content="HTML, débutant, tutoriel">
                    <meta name="author" content="Votre Nom">

<link> : Utilisée pour lier des feuilles de style externes, comme CSS.


                    <link rel="stylesheet" href="styles.css">

<script> : Pour inclure des scripts JavaScript. 


                    <script src="script.js"></script>

La partie <body>

    La balise <body> contient tout le contenu visible de la page, tel que les textes, images, vidéos, et autres éléments que les utilisateurs voient et interagissent avec. Voici quelques exemples d'éléments que l'on peut trouver dans la balise <body> :


Titres et paragraphes : Les titres sont définis par les balises <h1> à <h6>, et les paragraphes par la balise <p>.


                    <h1>Bienvenue dans le monde du HTML!</h1>
                    <p>Ceci est un paragraphe.</p>

Liens : Créés avec la balise <a>.


                    <a href="https://example.com">Cliquez ici</a>

Images : Incluses avec la balise <img>.


                    <img src="image.jpg" alt="Description de l'image">

Listes : Listes non ordonnées avec <ul> et listes ordonnées avec <ol>, chaque élément de liste étant défini par <li>.


                    <ul<
                        <li>Élément de liste non ordonnée</li>
                        <li>Autre élément</li>
                    </ul>
                    <ol>
                        <li>Premier élément</li>
                        <li>Deuxième élément</li>
                    </ol<

Formulaires : Utilisés pour collecter des données utilisateur.


                    <form action="/submit" method="post">
                        <label for="nom">;Nom :</label>
                        <input type="text" id="nom" name="nom">
                        <input type="submit" value="Envoyer">
                    </form>

    En résumé, la balise <head> est utilisée pour inclure des informations méta et des ressources externes, tandis que la balise <body> contient tout le contenu visible et interactif de la page
