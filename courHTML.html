Introduction a l'HTML

    L'HTML (HyperText Markup Language) est un langage informatique utilisé pour structurer et présenter du contenu sur le web. Créé en 1991 par Tim Berners-Lee, l'HTML est l'un des composants fondamentaux de l'Internet, permettant aux développeurs de créer des sites web interactifs et accessibles.

1. Structure de base d'une page HTML

Chaque document HTML commence par une déclaration de type :

<!DOCTYPE html>

Cette déclaration indique au navigateur que nous utilisons HTML5. Voici la structure de base d'une page HTML :


                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Ma première page HTML</title>
                    </head>
                    <body>
                        <h1>Bienvenue dans le monde du HTML!</h1>
                        <p>Ceci est un paragraphe.</p>
                    </body>
                    </html>

Le code ci-dessus crée une page web avec un titre "Bienvenue sur Mon Site Web!" et un paragraphe expliquant l'utilisation de l'HTML. Les navigateurs web interprètent le code HTML pour afficher la page de manière compréhensible pour les utilisateurs. L'HTML est souvent utilisé en combinaison avec d'autres technologies comme le CSS (Cascading Style Sheets) pour la mise en forme et le JavaScript pour l'interactivité. Ensemble, ces outils permettent de créer des sites web riches, dynamiques et esthétiques.

La partie <head>

    La balise <head> contient des méta-informations sur le document. Ces informations ne sont pas affichées directement sur la page mais sont cruciales pour les moteurs de recherche, les navigateurs et autres services web. Voici quelques éléments courants dans la balise <head> :


<title> : Le titre de la page, affiché dans l'onglet du navigateur.


                        <title>Ma première page HTML</title>

<meta> : Les balises meta fournissent des informations telles que le jeu de caractères utilisé, l'auteur de la page, ou des mots-clés pour les moteurs de recherche. 


                        <meta charset="UTF-8">
                        <meta name="description" content="Une introduction au HTML">
                        <meta name="keywords" content="HTML, débutant, tutoriel">
                        <meta name="author" content="Votre Nom">

<link> : Utilisée pour lier des feuilles de style externes, comme CSS.


                        <link rel="stylesheet" href="styles.css">

<script> : Pour inclure des scripts JavaScript. 


                        <script src="script.js"></script>

La partie <body>

    La balise <body> contient tout le contenu visible de la page, tel que les textes, images, vidéos, et autres éléments que les utilisateurs voient et interagissent avec. Voici quelques exemples d'éléments que l'on peut trouver dans la balise <body> :


Titres et paragraphes : Les titres sont définis par les balises <h1> à <h6>, et les paragraphes par la balise <p>.


                        <h1>Bienvenue dans le monde du HTML!</h1>
                        <p>Ceci est un paragraphe.</p>

Liens : Créés avec la balise <a>.


                        <a href="https://example.com">Cliquez ici</a>

Images : Incluses avec la balise <img>.


                        <img src="image.jpg" alt="Description de l'image">

Listes : Listes non ordonnées avec <ul> et listes ordonnées avec <ol>, chaque élément de liste étant défini par <li>.


                        <ul<
                            <li>Élément de liste non ordonnée</li>
                            <li>Autre élément</li>
                        </ul>
                        <ol>
                            <li>Premier élément</li>
                            <li>Deuxième élément</li>
                        </ol<

Formulaires : Utilisés pour collecter des données utilisateur.


                        <form action="/submit" method="post">
                            <label for="nom">;Nom :</label>
                            <input type="text" id="nom" name="nom">
                            <input type="submit" value="Envoyer">
                        </form>

    En résumé, la balise <head> est utilisée pour inclure des informations méta et des ressources externes, tandis que la balise <body> contient tout le contenu visible et interactif de la page

Les Balises

    Très bien ! Développons davantage sur les balises HTML. Les balises sont des éléments essentiels dans HTML car elles définissent la structure et le contenu de la page web. Chaque balise HTML commence par une balise ouvrante (<tagname>) et se termine par une balise fermante (</tagname>). Certaines balises, appelées balises auto-fermantes, n'ont pas de balise fermante.

Balises de titre

    Les balises de titre vont de <h1> à <h6>. Elles sont utilisées pour créer des titres de différentes tailles, <h1> étant le plus grand et <h6> le plus petit.


                        <h1>Titre principal</h1>
                        <h2>Sous-titre</h2>
                        <h3>Sous-sous-titre</h3>

Balise de paragraphe

    La balise <p> est utilisée pour définir des paragraphes de texte.


                        <p>Ceci est un paragraphe.</p>

Balise de lien

    La balise <a> est utilisée pour créer des liens hypertexte. L'attribut href spécifie l'URL vers laquelle le lien pointe.


                        <a href="https://example.com">Visitez notre site</a>

Balise d'image

    La balise <img> est utilisée pour afficher des images. Elle est auto-fermante et utilise les attributs src (source de l'image) et alt (texte alternatif pour les lecteurs d'écran).


                        <img src="image.jpg" alt="Description de l'image">

Balises de liste

    Les listes non ordonnées sont créées avec la balise <ul>, et les listes ordonnées avec <ol>. Chaque élément de liste est défini par la balise <li>.


                        <ul>
                            <li>Élément de liste non ordonnée</li>
                            <li>Autre élément</li>
                        </ul>
                        <ol>
                            <li>Premier élément de liste ordonnée</li>
                            <li>Deuxième élément</li>
                        </ol>

Balises de formulaire

    Les formulaires sont définis avec la balise <form>. Les éléments de formulaire comprennent les champs de texte (<input>), les zones de texte (<textarea>), les boutons de soumission (<button> ou <input type="submit">), et bien d'autres.


                        <form action="/submit" method="post">
                            <label for="nom">Nom :</label>
                            <input type="text" id="nom" name="nom">
                            <input type="submit" value="Envoyer">
                        </form>

Balises de tableau

    Les tableaux sont définis avec les balises <table>, <tr> (ligne de tableau), <th> (cellule d'en-tête), et <td> (cellule de données).


                        <table>
                            <tr>
                                <th>Nom</th>
                                <th>Âge</th>
                            </tr>
                            <tr>
                                <td>Jean</td>
                                <td>30</td>
                            </tr>
                            <tr>
                                <td>Marie</td>
                                <td>25</td>
                            </tr>
                        </table>

Balises sémantiques

    Les balises sémantiques donnent un sens structurel au contenu, ce qui améliore l'accessibilité et l'optimisation pour les moteurs de recherche. Quelques exemples incluent <header>, <footer>, <article>, <section>, et <nav>.


                        <header>
                            <h1>Entête de la page</h1>
                        </header>
                        <nav>
                            <ul>
                                <li><a href="#section1">Section 1</a></li>
                                <li><a href="#section2">Section 2</a></li>
                            </ul>
                        </nav>
                        <section>
                            <h2>Section principale</h2>
                            <p>Contenu de la section principale.</p>
                        </section>
                        <footer>
                            <p>Pied de page de la page</p>
                        </footer>
