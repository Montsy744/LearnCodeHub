Introduction a l'HTML

    L'HTML (HyperText Markup Language) est un langage informatique utilisé pour structurer et présenter du contenu sur le web. Créé en 1991 par Tim Berners-Lee, l'HTML est l'un des composants fondamentaux de l'Internet, permettant aux développeurs de créer des sites web interactifs et accessibles.

1. Structure de base d'une page HTML

Chaque document HTML commence par une déclaration de type :

<!DOCTYPE html>

Cette déclaration indique au navigateur que nous utilisons HTML5. Voici la structure de base d'une page HTML :

                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Ma première page HTML</title>
                    </head>
                    <body>
                        <h1>Bienvenue dans le monde du HTML!</h1>
                        <p>Ceci est un paragraphe.</p>
                    </body>
                    </html>

Le code ci-dessus crée une page web avec un titre "Bienvenue sur Mon Site Web!" et un paragraphe expliquant l'utilisation de l'HTML. Les navigateurs web interprètent le code HTML pour afficher la page de manière compréhensible pour les utilisateurs. L'HTML est souvent utilisé en combinaison avec d'autres technologies comme le CSS (Cascading Style Sheets) pour la mise en forme et le JavaScript pour l'interactivité. Ensemble, ces outils permettent de créer des sites web riches, dynamiques et esthétiques.

La partie <head>

    La balise <head> contient des méta-informations sur le document. Ces informations ne sont pas affichées directement sur la page mais sont cruciales pour les moteurs de recherche, les navigateurs et autres services web. Voici quelques éléments courants dans la balise <head> :


<title> : Le titre de la page, affiché dans l'onglet du navigateur.

                        <title>Ma première page HTML</title>

<meta> : Les balises meta fournissent des informations telles que le jeu de caractères utilisé, l'auteur de la page, ou des mots-clés pour les moteurs de recherche. 

                        <meta charset="UTF-8">
                        <meta name="description" content="Une introduction au HTML">
                        <meta name="keywords" content="HTML, débutant, tutoriel">
                        <meta name="author" content="Votre Nom">

<link> : Utilisée pour lier des feuilles de style externes, comme CSS.

                        <link rel="stylesheet" href="styles.css">

<script> : Pour inclure des scripts JavaScript. 

                        <script src="script.js"></script>

La partie <body>

    La balise <body> contient tout le contenu visible de la page, tel que les textes, images, vidéos, et autres éléments que les utilisateurs voient et interagissent avec. Voici quelques exemples d'éléments que l'on peut trouver dans la balise <body> :


Titres et paragraphes : Les titres sont définis par les balises <h1> à <h6>, et les paragraphes par la balise <p>.

                        <h1>Bienvenue dans le monde du HTML!</h1>
                        <p>Ceci est un paragraphe.</p>

Liens : Créés avec la balise <a>.

                        <a href="https://example.com">Cliquez ici</a>

Images : Incluses avec la balise <img>.

                        <img src="image.jpg" alt="Description de l'image">

Listes : Listes non ordonnées avec <ul> et listes ordonnées avec <ol>, chaque élément de liste étant défini par <li>.

                        <ul<
                            <li>Élément de liste non ordonnée</li>
                            <li>Autre élément</li>
                        </ul>
                        <ol>
                            <li>Premier élément</li>
                            <li>Deuxième élément</li>
                        </ol<

Formulaires : Utilisés pour collecter des données utilisateur.

                        <form action="/submit" method="post">
                            <label for="nom">;Nom :</label>
                            <input type="text" id="nom" name="nom">
                            <input type="submit" value="Envoyer">
                        </form>

    En résumé, la balise <head> est utilisée pour inclure des informations méta et des ressources externes, tandis que la balise <body> contient tout le contenu visible et interactif de la page

2. Les Balises

    Très bien ! Développons davantage sur les balises HTML. Les balises sont des éléments essentiels dans HTML car elles définissent la structure et le contenu de la page web. Chaque balise HTML commence par une balise ouvrante (<tagname>) et se termine par une balise fermante (</tagname>). Certaines balises, appelées balises auto-fermantes, n'ont pas de balise fermante.

Balises de titre

    Les balises de titre vont de <h1> à <h6>. Elles sont utilisées pour créer des titres de différentes tailles, <h1> étant le plus grand et <h6> le plus petit.

                        <h1>Titre principal</h1>
                        <h2>Sous-titre</h2>
                        <h3>Sous-sous-titre</h3>

Balise de paragraphe

    La balise <p> est utilisée pour définir des paragraphes de texte.

                        <p>Ceci est un paragraphe.</p>

Balise de lien

    La balise <a> est utilisée pour créer des liens hypertexte. L'attribut href spécifie l'URL vers laquelle le lien pointe.

                        <a href="https://example.com">Visitez notre site</a>

Balise d'image

    La balise <img> est utilisée pour afficher des images. Elle est auto-fermante et utilise les attributs src (source de l'image) et alt (texte alternatif pour les lecteurs d'écran).

                        <img src="image.jpg" alt="Description de l'image">

Balises de liste

    Les listes non ordonnées sont créées avec la balise <ul>, et les listes ordonnées avec <ol>. Chaque élément de liste est défini par la balise <li>.

                        <ul>
                            <li>Élément de liste non ordonnée</li>
                            <li>Autre élément</li>
                        </ul>
                        <ol>
                            <li>Premier élément de liste ordonnée</li>
                            <li>Deuxième élément</li>
                        </ol>

Balises de formulaire

    Les formulaires sont définis avec la balise <form>. Les éléments de formulaire comprennent les champs de texte (<input>), les zones de texte (<textarea>), les boutons de soumission (<button> ou <input type="submit">), et bien d'autres.

                        <form action="/submit" method="post">
                            <label for="nom">Nom :</label>
                            <input type="text" id="nom" name="nom">
                            <input type="submit" value="Envoyer">
                        </form>

Balises de tableau

    Les tableaux sont définis avec les balises <table>, <tr> (ligne de tableau), <th> (cellule d'en-tête), et <td> (cellule de données).

                        <table>
                            <tr>
                                <th>Nom</th>
                                <th>Âge</th>
                            </tr>
                            <tr>
                                <td>Jean</td>
                                <td>30</td>
                            </tr>
                            <tr>
                                <td>Marie</td>
                                <td>25</td>
                            </tr>
                        </table>

Balises sémantiques

    Les balises sémantiques donnent un sens structurel au contenu, ce qui améliore l'accessibilité et l'optimisation pour les moteurs de recherche. Quelques exemples incluent <header>, <footer>, <article>, <section>, et <nav>.

                        <header>
                            <h1>Entête de la page</h1>
                        </header>
                        <nav>
                            <ul>
                                <li><a href="#section1">Section 1</a></li>
                                <li><a href="#section2">Section 2</a></li>
                            </ul>
                        </nav>
                        <section>
                            <h2>Section principale</h2>
                            <p>Contenu de la section principale.</p>
                        </section>
                        <footer>
                            <p>Pied de page de la page</p>
                        </footer>

3. Les Attributs

    Parlons maintenant des attributs en HTML. Les attributs fournissent des informations supplémentaires sur les éléments HTML. Ils apparaissent dans la balise ouvrante et sont constitués d'un nom et d'une valeur séparés par un signe égal. Voyons quelques attributs courants :

Attributs communs

    id : Attribut unique qui identifie un élément spécifique de la page. Utile pour les styles CSS et les scripts JavaScript.

                        <p id="intro">Ceci est un paragraphe avec un identifiant unique.</p>

    class :Attribut utilisé pour regrouper plusieurs éléments sous une même classe. Utile pour appliquer des styles CSS de manière cohérente.

                        <p class="important">Ceci est un paragraphe important.</p>
                        <p class="important">Ceci est un autre paragraphe important.</p>

    style : Attribut utilisé pour ajouter des styles CSS directement à un élément.

                        <p style="color": red;">Ceci est un paragraphe en rouge.</p>

    title : Attribut utilisé pour fournir des informations supplémentaires affichées sous forme de tooltip lorsque l'utilisateur passe la souris sur l'élément.

                        <p title="Ce texte apparaîtra lorsque vous passerez la souris.">Survolez-moi !</p>

Attributs spécifique a certaine balise

    Balise <a> (Lien hypertexte) :

href : URL de la page vers laquelle le lien pointe.

target : Spécifie où ouvrir le document lié (_blank pour ouvrir dans un nouvel onglet).

                        <a href="https://example.com" target="_blank">Visitez notre site</a>

    Balise <img> (Image) :

src : Chemin de l'image.

alt : Texte alternatif pour les lecteurs d'écran.

width et height : Dimensions de l'image.

                        <img src="image.jpg" alt="Description de l'image" width="300" height="200">

    Balise <input> (Élément de formulaire) :

type : Type de champ (text, password, submit, etc.).

name : Nom du champ, utilisé lors de la soumission du formulaire.

placeholder : Texte d'exemple affiché à l'intérieur du champ.

                        <input type="text" name="nom" placeholder="Entrez votre nom">

Attribut globaux

    Les attributs globaux peuvent être utilisés sur n'importe quelle balise HTML pour fournir des fonctionnalités supplémentaires :

    data-* : Attributs personnalisés utilisés pour stocker des données supplémentaires sur un élément. 

                        <div data-info="informations supplémentaires">Contenu</div>

    hidden : Attribut booléen qui masque un élément.

                        <p hidden>Ce texte est caché.</p>

    lang : Attribut qui spécifie la langue de l'élément.

                        <p lang="fr">Ce paragraphe est en français.</p>

    tabindex : Attribut qui spécifie l'ordre de tabulation pour la navigation au clavier.

                        <button tabindex="1">Premier</button>
                        <button tabindex="2">Deuxième</button>

    Les attributs HTML sont essentiels pour ajouter des fonctionnalités et des informations supplémentaires aux éléments HTML. Ils permettent de personnaliser et d'interagir avec les éléments de manière plus avancée.

4. Les formulaire

    Développons la partie sur les formulaires en HTML. Les formulaires sont essentiels pour collecter des données auprès des utilisateurs, que ce soit pour des inscriptions, des commentaires, des recherches ou d'autres interactions sur une page web.

Un formulaire en HTML est créé à l'aide de la balise <form>. Cette balise contient plusieurs éléments de formulaire comme des champs de texte, des boutons, des cases à cocher, etc.

Voici un exemple de structure de base d'un formulaire :

                        <form action="/submit" method="post">
                            <label for="nom">Nom :</label>
                            <input type="text" id="nom" name="nom">
                            <input type="submit" value="Envoyer">
                        </form>

Attributs importants de la balise <form>

action : Spécifie l'URL où les données du formulaire seront envoyées lorsqu'il est soumis.

method : Indique la méthode HTTP à utiliser pour envoyer les données (GET ou POST).

    Champ de texte

Le champ de texte est créé avec la balise <input type="text">. Il permet à l'utilisateur de saisir des informations textuelles.

                        <label> for="nom">Nom :</label>
                        <input type="text" id="nom" name="nom">

    Champ de mot de passe

Le champ de mot de passe masque le texte saisi par l'utilisateur pour des raisons de sécurité.

                        <label for="motdepasse">Mot de passe :</label>
                        <input type="password" id="motdepasse" name="motdepasse">

    Bouton de soumission

Le bouton de soumission envoie les données du formulaire à l'URL spécifiée dans l'attribut action.

                        <input type="submit" value="Envoyer">

    Cases à cocher

Les cases à cocher permettent à l'utilisateur de sélectionner une ou plusieurs options indépendamment.

                        <label for="newsletter">S'abonner à la newsletter :</label>
                        <input type="checkbox" id="newsletter" name="newsletter">

    Boutons radio

Les boutons radio permettent à l'utilisateur de sélectionner une seule option parmi un groupe.

                        <label for="genre">Genre :</label>
                        <input type="radio" id="homme" name="genre" value="homme">
                        <label for="homme">Homme</label>
                        <input type="radio" id="femme" name="genre" value="femme">
                        <label for="femme">Femme</label>

    Liste déroulante

La liste déroulante permet à l'utilisateur de choisir une option parmi plusieurs.

                        <label for="pays">Pays :</label>
                        <select id="pays" name="pays">
                            <option value="france">France</option>
                            <option value="espagne">Espagne</option>
                            <option value="italie">Italie</option>
                        </select>

    Zone de texte

La zone de texte permet à l'utilisateur de saisir des paragraphes de texte.

                        <label for="message">Message :</label>
                        <textarea id="message" name="message"></textarea>

Attributs des éléments de formulaire

name : Nom du champ utilisé lors de la soumission du formulaire.

value : Valeur par défaut du champ.

placeholder : Valeur par défaut du champ.

required : Indique que le champ est obligatoire.

readonly : Rend le champ non modifiable.

disabled : Désactive le champ.

Exemple de formulaire complet

Voici un exemple de formulaire complet intégrant plusieurs éléments :

                        <form action="/submit" method="post">
                            <label for="nom">Nom :</label>
                            <input type="text" id="nom" name="nom" placeholder="Entrez votre nom" required>
                          
                            <label for="motdepasse">Mot de passe :</label>
                            <input type="password" id="motdepasse" name="motdepasse" required>
                          
                            <label for="newsletter">S'abonner à la newsletter :</label>
                            <input type="checkbox" id="newsletter" name="newsletter">
                          
                            <label for="genre">Genre :</label>
                            <input type="radio" id="homme" name="genre" value="homme">
                            <label for="homme">Homme</label>
                            <input type="radio" id="femme" name="genre" value="femme">
                            <label for="femme">Femme</label>
                          
                            <label for="pays">Pays :</label>
                            <select id="pays" name="pays">
                                <option value="france">France</option>
                                <option value="espagne">Espagne</option>
                                <option value="italie">Italie</option>
                            </select>
                          
                            <label for="message">Message :</label>
                            <textarea id="message" name="message"></textarea>
                          
                            <input type="submit" value="Envoyer">
                        </form>

5. Les Tableau

Développons davantage sur les tableaux en HTML. Les tableaux sont utilisés pour organiser des données tabulaires sur une page web. Ils sont définis à l'aide de plusieurs balises HTML spécifiques.

Structure de base d'un tableau

La balise <table> définit le tableau. À l'intérieur de cette balise, les lignes du tableau sont définies par des balises <tr> (table row). Chaque ligne contient des cellules de données définies par <td> (table data) et des cellules d'en-tête définies par <th> (table header).

Voici un exemple de structure de base d'un tableau :

                        <table>
                            <tr>
                                <th>Nom</th>
                                <th>Âge</th>
                            </tr>
                            <tr>
                                <td>Jean</td>
                                <td>30</td>
                            </tr>
                            <tr>
                                <td>Marie</td>
                                <td>25</td>
                            </tr>
                        </table>

Balises et attributs des tableaux

<table>

La balise <table> contient l'ensemble du tableau.

<tr>

La balise <tr> définit une ligne dans le tableau.

<th>

La balise <th> définit une cellule d'en-tête, généralement située dans les lignes ou colonnes en tête du tableau. Le texte dans une cellule est automatiquement mis en gras et centré.

<td>

La balise <td> définit une cellule de données standard dans le tableau.

Attributs courants

border : Ajoute une bordure autour des cellules du tableau.

                        <table border="1">
                            ...
                        </table>

colspan : Fusionne plusieurs colonnes pour une cellule spécifique.

                        <td colspan="2">Cellule fusionnée</td>

rowspan : Fusionne plusieurs lignes pour une cellule spécifique.

                        <td rowspan="2">Cellule fusionnée</td>

Sous-éléments de tableau

La balise <thead> est utilisée pour regrouper les lignes d'en-tête du tableau.

                        <table>
                            <thead>
                                <tr>
                                <th>Nom</th>
                                <th>Âge</th>
                                </tr>
                            </thead>
                        </table>

                        <table>

                                    <td>30</td>
                                </tr>

                                    <td>Marie</td>

                        <table>
                            <tfoot>
                                <tr>
                                    <td colspan="2">Fin du tableau</td>
                                </tr>
                            </tfoot>

                                }

                                border: 1px solid black;
                                padding: 8px;

                                }

                                }

                        </head>
                        <body>

                            <table>

                                        <td colspan="2">Fin du tableau</td>

                        </body>
